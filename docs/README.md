## Overview

CusOwn is a production-ready, multi-sector slot booking platform built with Next.js and Supabase.

## Key Features

- Multi-tenant business management
- Real-time slot booking
- Customer and owner dashboards
- Analytics and reporting
- Rescheduling and no-show handling
- Multi-channel notifications (WhatsApp)
- Security hardening
- Production-ready infrastructure
